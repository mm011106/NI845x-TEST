### NI-8451 を試してみました。
32bit送信を想定して4バイトのデータを連続して投げることを試しています。

1MBPSの設定です。極性などはサンプルプログラムのままなので、細かく追っていません。

1ms間隔で32bitのチャンクを10回送信しています。
